This project is a Pizza Sales Analysis Dashboard built using Power BI. The dataset contains detailed transactional information about pizza orders, including order ID, pizza name, category, size, ingredients, order date & time, quantity, unit price, and total price (quantity * unit price).
                    <br><br>The purpose of this project is to analyze sales performance and customer preferences by combining raw transactional data into an interactive dashboard. It highlights key performance indicators (KPIs) such as total revenue, total pizzas sold, total orders placed, average order value, and average pizzas per order.
                    <br><br>The goal is to provide a clear, data-driven view of pizza sales performance that helps the business understand what sells, when it sells, and how to maximize revenue while reducing losses from underperforming items.
